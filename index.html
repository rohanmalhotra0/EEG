Slide 1 Purpose & Problem

Turning Barriers into Opportunities

Purpose

  • We build lightweight, low‑cost tools so children with physical disabilities can interact online with ease and independence.

Problem

  • Many assistive devices cost thousands of dollars.
  • Mouse/keyboard assumes fine motor control.
  • Teachers lack affordable, scalable resources.

Slide 4 Solution

What is Green Sticker

What is Green Sticker

  • Chrome extension interprets a green sticker (computer vision).
  • Voice commands replace clicks.
  • On‑screen keyboard enables typing without hardware.
Flow: Webcam → Sticker Detection → Cursor Overlay → Voice/Keyboard → Website
